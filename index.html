  2. Respostas das perguntas do item "2)":
    1. Na tag <input> com atributo type="text", para que serve o atributo "placeholder"?
      • Resposta: Serve para exibir um texto dentro do campo de entrada, que serve como dica ou instrução para o usuário sobre o que deve ser digitado ali.

    2. Na tag <input> com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?
      • Resposta: Serve para já deixar a seleção selecionada quando o site for renderizado.

    3. Na tag <input> para que serve o atributo "value"?
      • Resposta: Ao definir o "value" você informa qual será o valor de cada checkbox ao ser pressionada.

    4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?
      • Resposta: Se insere o atributo "name" com o valor que seja o mesmo para todos os círculos de seleção que queira agrupar.

    5. Para que serve a tag <label>? Para que serve o atributo "for" nessa tag?
      • Resposta: Serve para relacionar o texto do label com uma checkbox, dessa forma, ao clicar no texto da label, você estará validando a checkbox que tenha o valor de "id" igual o valor do "for" dessa "label"

    6. Na tag <option> para que serve o atributo "selected"?
      • Resposta: Em uma lista de opções, o atributo "selected" serve para deixar uma opção pré-selecionada.

    7. Nas tags de formulário para que serve o atributo "required"?
      • Resposta:Esse atributo indica que um campo é obrigatório, e o não preenchimento dele impedirá o envio do formulário.

    8. Para que serve a tag <textarea>?
      • Resposta: Serve para criar uma área de texto multilinha, onde os usuários podem digitar grandes blocos de texto.

    9. Na tag <form> para que serve o atributo "action"?
      • Resposta: Define a URL para onde os dados do formulário serão enviados quando o formulário for enviado.

    10. Na tag <form> para que serve o atributo "method"?
      • Resposta: Serve para especificar o método HTTP usado para o envio do formulário, podendo ser "GET" ou "POST".

    11. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag <form>?
      • Resposta: O valor deve ser "POST", pois ao utilizar "GET", os dados do formulário apareceram na URL e podem ser visíveis na barra de endereços.

    12. Qual valor deve ser informado no atributo "method" na tag <form> em um formulário que será usado para fazer upload de arquivos?
      • Resposta: O valor do atributo "method" deve ser "POST" para upload de arquivos.

    13. O que as tags <fieldset> e <legend> produzem?
      • Resposta: As tags "fieldset" e "legend" são usadas para agrupar elementos de um formulário e criam uma legenda para o grupo. A tag "fieldset" cria um contorno ao redor dos elementos agrupados, e a tag "legend" fornece um título para esse grupo.
